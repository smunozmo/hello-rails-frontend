## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:smunozmo/hello-rails-frontend.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

### Testing

1. Run the tests
   ```sh
   npm test
   ```
   or
   ```sh
   npm run watch
   ```
2. See report in browser 
   ```sh
   open ./coverage/lcov-report/index.html
   ```
